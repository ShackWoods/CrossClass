# CrossClass
CrossClass is a program that allows you to define your data once and transpile matching classes for every language in your tech stack, ensuring there is one source of truth for your data models.

## Requirments
- Cmake >= v3.28.0
- Cunit

Development uses GCC. Any compiler that supports C17 should work, but your mileage may vary.

## Building From Source
- Run build.sh in the .ci directory
